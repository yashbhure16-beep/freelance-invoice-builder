# 📄 Freelance Invoice Builder

A professional, lightweight, and fully responsive web application designed for freelancers to generate, customize, and download PDF invoices instantly. 

This project was built and optimized with a focus on clean UI, flawless PDF rendering, and smooth user experience.

## 🚀 Live Demo
**[https://yashbhure16-beep.github.io/freelance-invoice-builder/]**

## ✨ Key Features
* **Dynamic Itemized Billing:** Add, update, and remove invoice line items dynamically.
* **Real-time Auto-Calculations:** Instantly calculates subtotals, custom tax percentages, and grand totals.
* **Flawless PDF Generation:** Utilizes a background "ghost-clone" method to generate perfectly scaled A4 PDFs without any screen flashing, layout shifting, or text cut-offs.
* **Modern UI/UX:** Built with a clean, Glassmorphism-inspired interface and smooth animations.
* **Zero Layout Shift:** Highly optimized CSS to ensure the UI remains perfectly stable across devices and during PDF generation.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Variables, Flexbox/Grid, Animations)
* **Logic:** Vanilla JavaScript (ES6+)
* **Libraries:** [html2pdf.js](https://ekoopmans.github.io/html2pdf.js/) (for client-side PDF generation)
